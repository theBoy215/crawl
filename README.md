一些工作中遇到的爬虫问题, 写成通用的代码.

pdf爬虫脚本导入相应模块即可直接使用.

rar爬虫脚本导入相应模块后还需要一些相对应的工具.
    如 window下需要将winrar文件中的rar.exe与unrar.exe文件移动到C:\windows中                                                                                                           
       linux需要安装sudo apt-get install -y rar unrar                                                                                                                               
       mac需要安装brew install unrar
